# Roles

Roles define which dashboard artefacts they have access (can view) within the Dashboard.

## Standard Roles

### 1. Super Admin / Manager
*   Full access to all settings.
*   Can invite and manage other users.
*   Can configure system-wide settings (Roles, Data Sources, etc).
*   Can view all data without geographic restriction.

### 2. Dashboard Viewer / User
*   Read-only access to data.
*   Cannot change settings or invite users.
*   View is often restricted to a specific assigned area.

### 3. Editor (Optional)
*   Can manage content like Announcements.
*   Cannot manage users or system configurations.

:::info
Roles are often configurable. Check the **Roles** settings page to see the specific permissions enabled for your instance.
:::
