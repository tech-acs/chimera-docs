# Localisation

Tout dans le tableau de bord peut être localisé dans n'importe quelle langue. Cela inclut la langue de l'interface et les données.

La localisation de l'interface se fait via des fichiers de langue. Vous pouvez en savoir plus à ce sujet dans la documentation Laravel. À ce jour, nous fournissons les langues anglaise, française et portugaise.

La localisation des données peut être effectuée directement depuis la section Gestion du tableau de bord. Tous les champs localisables sont affichés dans un champ de saisie spécial qui ressemble à ce qui suit.

![Champs multilingues](/img/developer/advanced-topics/multilanguage-fields.png)

Par exemple, pour fournir les traductions françaises de ces valeurs, basculez votre langue en français en sélectionnant "FR" dans le sélecteur de langue.
"FR" sera affiché sur les champs localisables (figure ci-dessus). Ensuite, il vous suffit de saisir la traduction française et de sauvegarder. C'est tout !
