# Introduction

Dashboard Starter Kit est un package Laravel magnifiquement conçu qui offre une base solide pour votre prochain tableau de bord de gestion de recensement ou d'enquête. En plus des fonctionnalités de base telles que la connexion, l'inscription, la vérification des e-mails, l'authentification à deux facteurs, la gestion des sessions, etc., il propose des graphiques réactifs, des filtres basés sur des zones, des indicateurs cartographiques et bien plus encore.

Il est conçu avec Tailwind CSS et utilise largement Laravel Livewire.

Bien qu'il apporte beaucoup de fonctionnalités, les développeurs sont censés implémenter les indicateurs souhaités en fonction de leur propre structure de base de données.
