# Prochaines étapes

Nous espérons que votre installation s'est bien déroulée. Cela signifie que la prochaine chose à faire est de commencer à construire votre tableau de bord.

Veuillez naviguer vers l'URL où votre tableau de bord est hébergé. Vous devriez voir une page d'accueil qui ressemble à ceci :

![Page d'accueil](/img/developer/landing.png)

Cliquez sur le bouton 'Commencer' ou sur le lien 'Se connecter' (coin supérieur droit) et vous serez dirigé vers une page de connexion où vous pourrez vous connecter en utilisant les identifiants que vous avez créés lors de l'installation via la commande adminify.
