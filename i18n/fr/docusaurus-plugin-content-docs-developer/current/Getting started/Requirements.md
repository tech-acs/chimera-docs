# Exigences
Étant donné que le Dashboard Starter Kit est construit sur la dernière version de Laravel, il nécessite PHP version 8.2 ou supérieure. Toutes les autres exigences de Laravel sont également des exigences ici.

Principalement, assurez-vous que votre environnement répond aux minimums suivants :

- [Php 8.2](https://www.digitalocean.com/community/tutorials/how-to-install-php-8-1-and-set-up-a-local-development-environment-on-ubuntu-22-04)
- Toutes les extensions php [requises par Laravel](https://laravel.com/docs/deployment#server-requirements)
- [Redis](https://redis.io/docs/getting-started/installation/)
- [PostgreSQL (>= 14)](https://www.postgresql.org/download/)
- [PostGIS](https://postgis.net/install/)
- Matériel avec au moins 32 Go de RAM et 8 cœurs de processeur
