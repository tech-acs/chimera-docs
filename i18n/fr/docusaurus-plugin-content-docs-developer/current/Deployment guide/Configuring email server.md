# Configuration du serveur de messagerie

Par défaut, l'envoi d'e-mails est désactivé.

Divers e-mails peuvent être envoyés par le tableau de bord.

- Liens d'invitation

- Liens de récupération de mot de passe (mot de passe oublié), etc.

Pour pouvoir envoyer des e-mails depuis le tableau de bord, vous devez disposer d'un serveur de messagerie préparé.

Les configurations peuvent être saisies dans le fichier .env de production comme décrit [ici](/docs/developer/getting-started/configuration#mail)
