# Contribution

## Comment contribuer

Nous suivons une feuille de route de développement et discutons des fonctionnalités et modifications dans le forum de discussion.
Les pull requests sont les bienvenues tant qu'elles sont conformes à notre feuille de route.

## Standards de codage

Nous suivons le [**PSR-12 : Style de codage étendu**](https://www.php-fig.org/psr/psr-12) et tout code soumis doit s'y conformer.
