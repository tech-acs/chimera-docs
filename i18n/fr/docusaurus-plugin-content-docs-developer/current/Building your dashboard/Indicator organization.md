# Organisation des indicateurs

Les indicateurs peuvent être organisés en ensembles logiques appelés pages. Les pages, si elles sont publiées, apparaîtront comme des éléments de menu dans la barre de navigation supérieure. Vous pouvez ajouter un indicateur à plusieurs pages.

## Création de pages

Vous pouvez créer des pages en accédant au sous-menu des pages dans le menu de gestion.

Là, vous pouvez créer une page en fournissant un titre, une description et un statut de publication.

## Ordre des indicateurs (par page)
En utilisant la fonctionnalité d'édition de page, vous pouvez également contrôler l'ordre d'affichage des indicateurs inclus dans une page.
